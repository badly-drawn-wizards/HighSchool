Create an account and login

  1. Open Account → Register.
  2. Enter the details and press Done.
  3. Open Account → Login and fill in your details to login with your new account.

Create a new demo

  1. Login to your account if you have not already.
  2. Open Demo → Create.
  3. Enter the details and press Done to open the editor.
  4. Open Help → Help contents for help on the editor.
Note: Demos with availability set to public can be seen by anyone. Set it to private if you only want yourself to see it. If you want to work on a demo and show it when it is complete, you can change the availability later under File → Edit details in the editor.

Fork an existing demo

  1. Login to your account if you have not already.
  2. Select the demo that you want to fork from in the list.
  3. Open Demo → Fork.
  4. Enter the details and press Done to open the editor.

Edit your own demo

  1. Login to your account if you have not already.
  2. Select the demo that you want to edit in the list.
  3. Open Demo → Edit to open the editor.
Note: A demo must have been created through your account in order for you to edit it. If you want to edit someone else's demo, fork it instead.
